AETHERIA — Desktop Edition
==========================

A cinematic 3D first-person adventure game.

HOW TO RUN
----------
1. Install Node.js 18+ from https://nodejs.org
2. Open a terminal in this folder
3. Run:  npm install
4. Run:  npm start

The game opens in a desktop window. Click to capture your mouse.

CONTROLS
--------
WASD       — Move
Shift      — Sprint
Space      — Jump
Mouse      — Look around
Left-Click — Swing sword
ESC        — Release mouse

STORY
-----
1. Collect 8 glowing crystal orbs (blue light beams)
2. Find the Elder across the lake (golden beacon)
3. He gives you a sword — night falls
4. Kill 10 zombies with left-click swings
5. Save the village!
